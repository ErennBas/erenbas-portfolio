Cautomat - Akıllı Otomatik Mama Otomatı

Açıklama

  • Cautomat

Cautomat, bir arkadaşımın tombiş olma yolunda olan kedisi için tasarladığım, çizdiğim, kodladığım ve ürettiğim bir akıllı mama otomatıdır.

Bu projeyi geliştirirken frontend tarafında Angular 11 ve Ionic kullanarak Capacior aracılığıyla bir mobil uygulama yaptım. Bir backend tarafı yoktur.

Sistemin cihaz tarafı için ise C dilini kullandım. VS Code üzerinde Platform IO altyapısı ile geliştirdim. Devre kartını ise EasyEDA yazılımını kullanarak geliştirdim.

Teknolojiler

  • Angular 11
  • Ionic
  • Capacitor
  • Javascript
  • Node JS
  • Plaftorm IO
  • C / C++
  • EasyEDA
