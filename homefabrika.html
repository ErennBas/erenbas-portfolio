Home Fabrika, Dans Fabrikanın bir alt kuruluşu olan online dans eğitim platformudur.

Bu projeyi geliştirirken frontend tarafında Angular 11' i kullandım. Backend olarak PHP dilinde Slim Framework kullanarak Bir RestFull API geliştirdim. Sistemde Viemo altyapısı ile bir canlı yayın mekanizması var fakat bu mekanizmanın chat kısmını kullanmak içni kullanıcıların vimeo hesabının olması gibi saçma bir durumdan dolayı Node JS kullanarak kendi chat altyapımızı kurdum. Basit bir mikroservis mantığı ile chat sorununu bu şekilde çözdüm.

Sistem bir kullanıcı paneli ve bir de admin paneli ile birlikte 2 tip clienta sahiptir.

Teknolojiler

  • Javascript
  • Node JS
  • Angular 11
  • Socket.io
  • PHP 7.3
  • PHP Slim Framework 3
