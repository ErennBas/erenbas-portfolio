IscoBox - Daha Akıllı Kamera Sistemleri için IOT Cihaz

Açıklama

  • İscobox

  • iscobox.com

IscoBox, kibrit kutusu büyüklüğünde akıllı bir IOT cihazıdır. Bulunduğu ağdaki kamera sistemlerinin sağlık durumlarını raporlar, hataları tespit eder ve sizden onay aldıklarını düzeltir. Bu sayede güvenlik firmaları, kamera kurulumu yaptıkları tüm noktaları tek bir ekran üzerinden monitoring edebilir ve yönetebilir.

Bu projeyi geliştirirken frontend tarafında Vue 3 ve Quasar Framework kullanarak geliştirdim. Mobil Uygulama tarafında Cordova'dan faydalandım. Backend tarafında ise Node JS - Fastify Frameworkunden faydalandım. Bildirim sistemi için Google Firebase Cloud Messaging sistemini kullandım.

Sistemin cihaz tarafı için ise C dilini kullandım. VS Code üzerinde Platform IO altyapısı ile geliştirdim. Devre kartını ise EasyEDA yazılımını kullanarak geliştirdim.
