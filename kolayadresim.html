Kolayadresim - Dijital Kartvizit Uygulaması

Açıklama

  • Kolay Adresim

  • kolayadresim.com

Kolayadresim, telefonların arkasına yapıştırılabilen, araba camları, dükkan camları, anahtarlık, kedi-köpek tasması gibi size ulaşılması istediğiniz her yere yapıştırabileceğiniz nfc etiketi veya karekod ile paylaşılan dijital kartvizit sistemidir.

Bu projeyi geliştirirken frontend tarafında Vue 3 ve Quasar Framework kullanarak geliştirdim. Backend tarafında ise PHP Slim Framework 3 kullandım.

Teknolojiler

  • Vue 3
  • Quasar
  • Javascript
  • PHP Slim Framework 3
