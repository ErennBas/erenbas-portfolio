Tvarz App - Video Platformu

Açıklama

  • MenuExpress

  • tvarz.com

Tvarz, eski tip televizyon kanalının, internette kullanıcıyla buluşan interaktif dönüşümünün sonucu bir tarz olarak 2021’de Y kuşağının son gençleri tarafından hayata geçirilmiştir.

Bu projeyi geliştirirken frontend tarafında Angular 11' i kullandım. Backend olarak PHP dilinde Slim Framework kullanarak Bir RestFull API geliştirdim.

Teknolojiler

  • Javascript
  • Angular 11
  • PHP 7.3
  • PHP Slim Framework 3
